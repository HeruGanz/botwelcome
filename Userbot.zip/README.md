# Userbot Pyrogram
Userbot sederhana menggunakan Pyrogram dengan fitur:
- .ping
- .id
- .setmsg (dengan reply)
- .setgroup <url>
- .setdelay <detik>
- .startsend
- .stop

## Instalasi
1. `pip install -r requirements.txt`
2. Edit `config.py` isi API ID dan HASH dari https://my.telegram.org
3. Jalankan: `python main.py`
